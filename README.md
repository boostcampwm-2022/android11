## 🌟 주요 기능

### 🤙 약속 생성 및 참여
- 친구와 만날 일정이 생겼다면 약속을 생성하세요.
- 참여 코드를 통해 약속에 참여할 수 있어요.

### 📚 지난 약속 확인
- 이미 지나간 약속을 확인해요.
- 누가 제일 약속을 잘 지켰는지 확인할 수 있어요.

### 🗺 실시간 위치 공유
- 약속 일정이 시작되면 친구가 현재 어디 있는지 확인할 수 있어요.
- 실시간으로 순위를 확인할 수 있어요.

### 🌈 시간마다 변화하는 자기장
- 시간이 지날 수록 조여오는 자기장을 피해 도착하세요.
- 자기장에 닿으면 HP를 소모해요.
